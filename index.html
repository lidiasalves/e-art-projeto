Você sabe o que é um NFT?

É uma sigla “non-fungible token”, traduzido para o português: token não-fungivel. Em outras palavras um token no universo das criptomoedas é a representação de um ativo como dinheiro, propriedade ou obra de arte registrada em uma blockchain. Essa forma de negócio é recente, conquistou olhares no ano de 2014, através da obra de arte de Kevin McCoy, Quantum. De lá pra cá os NFT'S se tornaram verdadeiras tendências de colecionáveis no mercado de cripto! E é por essa razão que apresentaremos alguns artistas nesse modelo de arte digital.
